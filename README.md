# AI/ML Exploration Platform

This project uses TensorFlow, Keras, and OpenCV to build a handwritten digit recognition model.

## Features
- Image classification using deep learning
- Trained on MNIST dataset
- Predicts digits from custom images

## Tech Stack
- Python
- TensorFlow
- Keras
- OpenCV

## Results
Achieved ~95% accuracy on test data.

## Features

### Level 01
- Trained digit recognition model using TensorFlow

### Level 02
- Built web app using Streamlit
- Upload image and get prediction
